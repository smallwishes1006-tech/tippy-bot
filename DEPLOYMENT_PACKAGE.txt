```

```

### Step 2: Deploy Bot (1 min)
```
1. Render Dashboard → "New +" → "Worker"
2. Repository: smallwishes1006-tech/tippy-bot
3. Name: tippy-bot
4. Runtime: Docker
5. Plan: Free (shows $0/month)
6. Region: oregon (or closest to you)

Go to service "Environment" tab, add these 5 from your local .env file:

```

MASTER_WALLET_PRIVATE_KEY
[copy from your .env file]

BLOCKCYPHER_API_KEY
[copy from your .env file]

[copy from your .env file]

Click "Save" after each variable.

### Step 4: Verify Deployment (1 min)
```
Go to "Logs" tab

(shows when bot is fully online)
```

### Step 5: Test in Discord (30 sec)
```
In any Discord channel with the bot:

```

### Step 6: Stop Replit (30 sec)

## 🔧 TROUBLESHOOTING

**Bot doesn't come online:**
- Check Render logs for error messages
- Verify all 5 environment variables are set correctly
- Check Discord bot has "Send Messages" permission in server

**Bot goes offline:**
- Check Render logs for crash reason
- Verify Discord token is still valid
- Check internet connectivity

**Want to check logs:**
- Render Dashboard → Your Service → "Logs" tab
- Scroll down to see real-time output
